Problem description at Codewars can be found
[here](https://www.codewars.com/kata/5590961e6620c0825000008f/train/python).

-------------

Given a string, swap the case for each of the letters.
<br>

Examples:
```
""           -->   ""
"CodeWars"   -->   "cODEwARS"
"abc"        -->   "ABC"
"ABC"        -->   "abc"
"123235"     -->   "123235"
```
